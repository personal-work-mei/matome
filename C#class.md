## C#とは
* OOP(Object Oriented Programing)オブジェクト指向プログラミング

### システム開発の手順
1. システム分析
   要求の分析
   システムがしなければならないこと、やらなければならないことを明確にする。

2. システム設計
   分析した結果をシステム上稼働させるにはどうすればよいか検討

3. クラス設計
   クラス図、シーケンス図 レビュー

4. PG/テスト/デバッグ

### クラス設計
システムの仕様を詳細に文章化する。
名詞をクラス候補とする。

* 属性-そのクラスの成り立ちを示すデータ
* 操作-そのクラスに対する命令の受付窓口

### クラスの関連の十分な検討
* クラスの仕様範囲
  * この仕様は、どのクラスの仕様か
  * 何をするか
  * それをどのクラスが実施するか
  * PGしながら、テストパターン記述

## 値型と参照型
* 参照型
  * string
  * クラス
  * インターフェース
  * 配列
  * デリゲート

* 値型
  * int
  * long
  * doubleなど
  * 列挙型(enum)
  * 構造体(struct)
    * C++ -> 構造体は値型でも参照型でも両方アクセス可能
    * 大きいサイズを持ち、大量にコピーするようなものは構造体ではなくクラスを定義
